Five Style Directions for the Demonology HUD

These are overlay treatments of Blizzard's Cooldown Manager, not replacement UIs: every frame keeps the two vertical icon columns (Essential + Utility, priority top-down), Blizzard's radial cooldown wipe, the 0–5 Soul Shard rail, and two draining buff bars. Icons stay visible and recognizable — each direction restyles them with ops a WoW addon actually has (grayscale + vertex-color tint, border/backdrop textures, fontstrings, texture rotation, alpha/scale pulses, static overlay textures). Glyphs stand in for the real icon art. Shown states everywhere: TYRA about to come up, FELR mid-sweep, DARK on cooldown, CORE proc-glowing, shards capped at 5 with a glint.

A · Powered Goggles — Gnomeregan Mechatronics

Blizzard's icon columns set into brass instrument housings: riveted frames around each tile, icons tinted by ability-group hue, punched tickertape labels, amber readouts, a small clockwork chrono-dial ticking in each header plate, and a goggle-lens vignette over the whole scene.

WoW mapping: per-group icon tint = SetDesaturated(true)+SetVertexColor · brass frames/rivets/tape = border+backdrop textures · label/keybind/countdown = fontstrings · Blizzard's radial sweep kept on FELR/DARK · chrono-dial needle = texture SetRotation anim · lens vignette = one stretched semi-transparent overlay texture (fake curvature, no shader).

B · CRT / DOS Terminal

The whole Cooldown Manager rendered through a green phosphor monitor: every icon is literally the same Blizzard icon, desaturated and multiplied green — shape stays recognizable, everything reads as one tube. Text rows with live countdowns beside each tile; the Demonic Core proc is inverse-video blink; the shard cap is a blinking ✦.

WoW mapping: uniform green tint on ALL icons = SetDesaturated(true)+SetVertexColor(green) — the key technique, shown literally · all rows/meters/countdowns = fontstrings · Blizzard's sweep kept (visible under the green tint) · proc = our own blink via alpha anim · scanlines + vignette + flicker = stretched overlay textures + alpha animation (fake CRT, no curvature/bloom shader).

C · LCARS

Okudagram frame around the icon grid: each Blizzard icon gets a flat color keyline and a solid color bar when ready; on cooldown the bar hollows to an outline with a remaining-time fill while the icon keeps its sweep. Blockier than screen-accurate LCARS on purpose — the swoopy elbows are approximated with modest corner rounding.

WoW mapping: group tints on icons = SetDesaturated+SetVertexColor · flat bars/rail blocks = plain color textures · fills = StatusBar frames · text = fontstrings · Blizzard sweep kept · proc = alpha-blink on the bar · ⚠ true LCARS elbow curves need bespoke corner-texture art — mocked here with simple rounding, flagged as an art dependency.

D · The Bound Grimoire

The icon columns as entries in a living demonic manuscript: each Blizzard icon sits inside an inked cartouche, tinted by its ink (fel-green burst, shadow-violet spend); ready entries glow, cooldown entries dry out while a luminous ink line refills along the base. A faint summoning circle turns slowly behind the page. Soul-gem diamonds flare gold and sparkle at cap.

WoW mapping: ink tints on icons = SetDesaturated+SetVertexColor · cartouches = static hand-inked nine-slice border textures (⚠ the wobble is baked art, not a freeform vector anim) · glow/dry = alpha+glow pulse · refill line = thin StatusBar · summoning circle = large texture with SetRotation anim · Blizzard sweep kept · gems = rotated square textures.

E · Neon District — Cyberpunk / Synthwave

The icon grid behind rain-slick glass: dark translucent row plates, each Blizzard icon rimmed in its group's neon (cyan burst, magenta spend), chromatic-fringed labels, a perspective grid burning on the horizon, and an occasional glitch twitch on the column headers. Shards are skewed neon cells that flip gold at cap.

WoW mapping: neon group tints = SetDesaturated+SetVertexColor · rim glow = border texture + alpha pulse · glass plates = semi-transparent backdrops · chromatic fringe = two offset tinted fontstring copies (fake aberration) · grid horizon = one static overlay texture · glitch = brief SetPoint/alpha jiggle · Blizzard sweep kept · no real shaders anywhere.
