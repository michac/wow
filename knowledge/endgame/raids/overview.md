# Midnight Season 1 Raids

Season 1 has **three raids** (9 bosses total) plus a 4-boss world-boss
rotation. Names and boss lists below are from the in-game journal via the
Blizzard API — they correct earlier guide-derived notes that described
"two raids".

| Raid | Journal ID | Bosses | File |
|------|-----------|--------|------|
| The Voidspire | 1307 | 6 | `the-voidspire.md` |
| The Dreamrift | 1314 | 1 | `the-dreamrift.md` |
| March on Quel'Danas | 1308 | 2 | `march-on-quel-danas.md` |
| Midnight (world bosses) | 1312 | 4 rotating | below |

Warcraft Logs tracks all nine bosses in a single zone (46, "VS / DR / MQD").
