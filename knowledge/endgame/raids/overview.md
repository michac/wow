# Midnight Season 1 Raids

As of patch **12.0.7 "Revelations"** (live 2026-06-16) Season 1 has **four
raids** (10 bosses total) plus a 4-boss world-boss rotation. The added raid is
**Sporefall** (single-boss, vs Rotmire). Names and boss lists below are from
the in-game journal via the Blizzard API — they correct earlier guide-derived
notes that described "two raids".

| Raid | Journal ID | Bosses | File |
|------|-----------|--------|------|
| The Voidspire | 1307 | 6 | `the-voidspire.md` |
| The Dreamrift | 1314 | 1 | `the-dreamrift.md` |
| March on Quel'Danas | 1308 | 2 | `march-on-quel-danas.md` |
| Sporefall | TBD | 1 (Rotmire) | `sporefall.md` |
| Midnight (world bosses) | 1312 | 4 rotating | below |

Warcraft Logs zone 46 ("VS / DR / MQD") tracks the original nine bosses;
Sporefall (added in 12.0.7) is a separate single-boss instance.

## Sporefall (12.0.7)

Single-boss raid in **Harandar** against the fungal giant **Rotmire**.
Available in **Raid Finder / Normal / Heroic / Mythic**. Notably, Mythic is
the game's first **flexible Mythic** (scales **15–25** players) rather than
fixed 20.

- **Sporefused gear** by difficulty: RF **259** · Normal **272** · Heroic
  **285** · Mythic **298**.
- No tier set / class weapons drop here.
- Collectibles: **Luminous Sporeglider** mount (earned by collecting 4
  Delicious Sporesnacks from weekly Rotmire kills), **Luminous Rotshroom**
  housing decor, plus toys/cosmetics/achievement rewards.
- RF entry requirement: level 90, ~ilvl 240.

See `sporefall.md` for boss mechanics.
