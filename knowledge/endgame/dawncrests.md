# Dawncrests (Midnight Season 1 upgrade currency)

Tiers: **Adventurer / Veteran / Champion / Hero / Myth**.

## Weekly cap

- **100 crests per tier per week** (= 5 upgrades per track per week).
- Cap applies to normal earning (ritual sites, delves, M+, raid).

## Uncapped / cap-exempt sources (don't skip these)

- **Cracked Keystone quest**: 20 Hero + 20 Myth, cap-exempt.
- **Nullius kill (Torment Rise delve)**: crests don't count toward cap.
- Nightmare Prey weekly was cap-exempt but **hotfixed to count**.
- (Wowhead/Icy Veins maintain running lists of uncapped sources —
  re-check around content updates.)

## Earning-rate reference

- Ritual sites T5: 20 Hero/run → **~5 runs/week saturates the Hero cap**.
- Delve T11 gilded stashes: 7 Myth × 3/week = 21 (see `delves/overview.md`).
- M+9s: Myth crests per run (cap applies).
